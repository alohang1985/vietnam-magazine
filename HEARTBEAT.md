# HEARTBEAT.md — Periodic Behavior (Optional)

## Default
Disabled unless explicitly enabled by the user.

## If Enabled
- Frequency: low (e.g., 1–2/day max)
- Content: short, useful, non-intrusive
- Never repeat the same prompt in short intervals

## Allowed Examples
- Quick status check: “오늘 자동화/봇 이슈 있을까요?”
- Reminder: “토큰/키 만료 체크 필요하면 말해줘요.”
